## 🎨 Personnaliser en 5 minutes

### Changer le titre
```html
Trouvez: <title>Extra Now - Trouvez un Extra Qualifié en 24h</title>
Remplacez par votre titre
```

### Changer les couleurs
```css
Trouvez: --primary: #1a1a2e;
Remplacez par votre couleur
Exemple: --primary: #your-color;
```

### Changer le contenu
```html
Trouvez: <h1>Trouvez un extra qualifié en 24h</h1>
Remplacez par votre texte
```

### Ajouter une section
```html
Copiez une section complète
Collez-la à l'endroit voulu
Modifiez le contenu
```

## 📱 Tester sur mobile

### Dans Chrome/Firefox
1. Ouvrir le site
2. Appuyez sur F12 (ou Cmd+Option+I sur Mac)
3. Cliquez sur l'icône "mobile" (coin haut-gauche)
4. Testez la responsivité

### Dans un téléphone réel
1. Envoyez-vous le fichier par email
2. Ouvrez-le dans le navigateur du téléphone
3. Testez la navigation et formulaires
