# 📘 EXTRA NOW - Plateforme de Mise en Relation d'Extras

## 🚀 À Propos du Projet

**Extra Now** est une plateforme digitale innovante qui connecte :
- **Restaurateurs, hôteliers, traiteurs, organisateurs d'événements** qui ont besoin d'extras rapidement
- **Serveurs, cuisiniers, barmans, plongeurs, hôtesses** qui cherchent des missions flexibles

## 🌐 PAGES INCLUSES

Le fichier `extra-now-site.html` contient 4 pages complètes et fonctionnelles :

### 1️⃣ **Accueil** (Page par défaut)
- Hero impressionnant avec CTA dual
- Section "Comment ça marche" (3 étapes)
- Statistiques clés
- Témoignages clients
- Garantie & Sécurité
- Call-to-action final
- Footer complet

### 2️⃣ **Page Entreprises**
- Hero dédié aux entreprises
- 4 avantages clés (gain de temps, personnel qualifié, flexibilité, simplicité)
- Formulaire rapide pour demander un extra
- Champs : type de mission, date, lieu, nombre d'extras

### 3️⃣ **Page Extras**
- Hero dédié aux extras
- 4 bénéfices (liberté, missions régulières, paiement rapide, réputation)
- 3 étapes simples d'inscription
- CTA d'inscription gratuite

### 4️⃣ **Page FAQ**
- Questions pour entreprises
- Questions pour extras
- Réponses détaillées et claires
